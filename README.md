# The Mirrored Spaceman

Text-based game "The Mirrored Spaceman" (Russian: "Зеркальный космонавт") by Evgeniy Tugolukov (Russian: Евгений Туголуков), a.k.a. Korwin.

The game is in Russian, developed for FireURQ, and adapted for [UrqW](https://github.com/urqw/UrqW) text-based game engine.

This repository contains the source code and other resources of the game. The game is available for launch in the [UrqW engine online catalog](https://urqw.github.io/UrqW/#mirrored_spaceman).

The game was first published on January 30, 2013.

Interactive fiction identifier (IFID) of the game which is its MD5 hash: 2001EC444DBA38EC8EEB6A6FF856D88E

## Description

Браслет-коммуникатор прервал твой сон немелодичной трелью. "Начальство", - подумал ты, - "Соединить". Появилось голографическое изображение знакомого лица. На связи был Командор Биг.\
- Привет, Виктор! Ты мне нужен. Я сегодня жду тебя там, где мы отмечали твоё повышение. Не опаздывай...\
- Хорошо, Биг. Я понял...
